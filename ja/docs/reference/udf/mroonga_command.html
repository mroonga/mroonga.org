5.5.1. mroonga_command()¶

バージョン 3.02 で追加.

5.5.1.1. 概要¶

mroonga_command() UDFは与えられた文字列をGroongaコマンドとして実行し、Groongaコマンドの結果を返します。おそらく、Groongaコマンドは、MySQLのクエリーより高速です。

mroonga_command() は上級者向けのUDFです。通常このUDFを使う必要はありません。

5.5.1.2. 構文¶

mroonga_command() の引数は1つで必須の引数です。:

mroonga_command(command)

command は文字列型の値です。実行するGroongaコマンドを指定します。
