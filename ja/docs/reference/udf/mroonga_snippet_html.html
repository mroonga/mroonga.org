5.5.7. mroonga_snippet_html()¶

バージョン 5.09 で追加.

5.5.7.1. 概要¶

mroonga_snippet_html() UDFは、コンテキスト内のキーワードをハイライトする機能を提供します。

注釈

この機能は実験的です。引数や引数の値が予告なく変更される可能性があります。

5.5.7.2. 構文¶

mroonga_snippet_html() には必須引数と省略可能引数とがあります:

mroonga_snippet_html(document, key_word)

5.5.7.3. 使いかた¶

5.5.7.4. 引数¶

5.5.7.4.1. 必須引数¶

多くの必須引数があります。

5.5.7.4.2. 省略可能引数¶

5.5.7.5. 返り値¶

スニペット文字列を返します。
