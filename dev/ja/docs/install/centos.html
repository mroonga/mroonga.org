CentOS#

CentOS 7がEOLになったのでMroonga 14.04以降はCentOS 7用パッケージを提供していません。

Mroongaは、AlmaLinux 8をサポートしているので、CentOS 7からAlmaLinux 8にOSを切り替えることで引続きMroongaを利用できます。

AlmaLinuxを参照してください。
